# 🎉 Avija Business platform

A full-stack Event Management web application built using **React, Vite, Node.js, Express.js, and MongoDB**. The application allows users to explore event management services, contact the company, and enables administrators to manage events through a secure admin panel.

---

## 🚀 Features

### 👥 User Features

* Beautiful responsive homepage
* Professional Services page
* Portfolio showcasing recent events
* Event Details page
* Contact Us page
* Contact form connected to MongoDB
* Smooth navigation using React Router

### 🔐 Admin Features

* Admin Login
* Secure Dashboard
* View customer enquiries
* Add New Events
* Logout functionality

---

## ⚙️ Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Frontend

```bash
cd avija
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

```
MONGO_URL=Your MongoDB Connection String
```
